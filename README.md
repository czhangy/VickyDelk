## Getting Started

<!-- Prerequisites -->

### Prerequisites

This project uses NPM as a package manager

<!-- Run Locally -->

### Run Locally

Clone the project

```bash
  git clone https://github.com/czhangy/vickydelk.git
```

Install dependencies

```bash
  npm install
```

Start the application

```bash
  npm run dev
```
